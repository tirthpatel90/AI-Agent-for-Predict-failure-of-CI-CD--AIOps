# AutoHeal CI - AI-Powered CI/CD Predictive Platform

AutoHeal CI is an industry-level, AI-driven CI/CD platform designed to predict deployment failures, automatically suggest fixes, and provide comprehensive insights into pipeline health. It features a premium "Dark Glass" UI/UX with fluid, physics-based animations.

## Key Features
- **Predictive Failure Analysis:** Monitors commits and build parameters to predict the probability of failure before deployment.
- **Auto-Fix Engine:** Automatically suggests or applies fixes using a growing AI knowledge base of past failures and resolution strategies.
- **Interactive Pipeline Visualizer:** Beautiful, node-based interactive flow diagrams to monitor CI/CD stages in real-time.
- **Learning & Growth Metrics:** Visualizes how the AI engine improves its accuracy and expands its custom error knowledge base over time.
- **Premium Glassmorphism UI:** Built with Framer Motion and Tailwind CSS v4, featuring MacOS-style dock navigation, custom glow gradients, animated charts, and fluid transitions.

## Technology Stack
- **Framework:** Next.js 16 (App Router + Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion (Spring Physics & Layout Id animations)
- **Data Visualization:** Recharts
- **Icons:** Lucide React

## Getting Started
To run the development server locally:

```bash
# Install dependencies
npm install

# Start the local development server (Turbopack)
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the complete AutoHeal CI platform.
